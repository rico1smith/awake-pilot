Awake-Pilot

Driver Fatigue Detection System | MSU Capstone Project

About the Project

Awake-Pilot is an AI-driven driver monitoring system that helps prevent accidents caused by drowsy driving.

Key Features

  • Jetson Nano for real-time processing
  • Facial and eye movement detection
  • Infrared + RGB camera integration
  • Audio and seat vibration alerts
